# BayesPLCox

Bayesian inference for Cox proportional hazards models via rank-ordered likelihoods.

## Overview

**BayesPLCox** implements Bayesian Cox regression based on Plackett–Luce (PL) and generalized Plackett–Luce (GPL) likelihoods.

## Associated paper

This package accompanies:

> Ohigashi T, Orihara S, Sugasawa S.  
> *Bayesian Inference in the Cox Model via Rank-Ordered Likelihood.*
